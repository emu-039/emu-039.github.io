「The Cool World」WEBサイト制作

概要

架空の写真家Daisy の写真展"The Cool World"を宣伝し興味を持った人に会場へ来てもらうために、特設のWEBサイトを制作しました。

ターゲット

  • 写真撮影が趣味の人
  • カッコいい写真が好きな人
  • 外国人の被写体に興味がある人
工夫点

写真展の宣伝サイトであることをアピールするために、トップページにはタイトルをイメージした写真を複数枚組み合わせた画像を表示しました。また、ベースカラーはブラック、メインカラーはロイヤルパープル、アクセントカラーはイエローにし、暗い雰囲気の中でも少し鮮やかさを持たせることで、ターゲットの印象に残るよう色味を調整しました。

制作期間

２週間
制作人数

１人
使用言語・ツール

HTML / CSS / JavaScript / Figma
